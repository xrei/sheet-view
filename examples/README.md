# Examples

Source for the docs site's live demos — the VitePress panels import these files
directly and show them as snippets. There's no standalone app to run; use
`pnpm docs:dev` from the repo root to see them.

- `vanilla/demos/` — the framework-agnostic core on plain DOM (`basic.js`, `themed.js`).
- `vanilla/app.js` + `dom.js` — the colour-coded anatomy reference.
- `react/src/demos/` — the `sheet-view/react` adapter, one component per panel.
